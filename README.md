## Installation

### Linux

**Requirements**

- `make`

**Process**

* Download the `*.tar.xz` file from the [releases page](https://github.com/thomas-marquis/s3-box/releases)
* Extract the archive (e.g. `tar -xf s3-box-*.tar.xz`)
* Run `make user-install`
